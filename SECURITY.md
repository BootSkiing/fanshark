# Security & Responsible Use

`fanshark` is a **defensive-research proof-of-concept**. It measures whether an
inference host emits a recoverable acoustic side-channel signal. It does not,
and is not designed to, recover model weights.

## Authorized use only

Only capture and analyze systems that **you own** or are **explicitly
authorized** to assess (e.g., under a signed penetration-testing engagement or
in a lab you control). Recording third-party hardware without permission may be
illegal in your jurisdiction.

## Responsible disclosure

If you use this tooling to identify a real leakage issue in a product, follow
coordinated disclosure with the affected vendor (a 90-day timeline is a common
default) before any public discussion.

## Reporting issues in fanshark itself

Open a private security advisory on the repository, or contact the maintainers
before filing a public issue for anything sensitive.
